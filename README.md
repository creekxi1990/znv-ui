# znv-ui

一套基于vue2.0的UI组件库，提供常用组件和公共样式、函数

## Docs

[document]()

## install 安装

```bash
npm install znv-ui -S
```

## use 使用

使用静态引入

```
<!-- import stylesheet -->
<link rel="stylesheet" href="znv-ui/styles/index.css">
<!-- import znv-ui -->
<script src="znv-ui/znv-ui.umd.min.js"></script>
```

使用

```vue
<template>
    <znv-button>测试按钮</znv-button>
</template>
```
